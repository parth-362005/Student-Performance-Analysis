# Student-Performance-Analysis
Student Performance Analysis and Score Prediction: Analyzed 1,000 student records using Python, Pandas, and Seaborn in Google Colab. Performed data cleaning, EDA, and visualization, and built a Random Forest model to predict scores, uncovering key factors influencing academic performance.
# 📊 Student Performance Analysis and Score Prediction

## 📌 Project Overview
This project analyzes student academic performance using data analytics and machine learning techniques. It identifies key factors affecting student success and predicts exam scores using Python.

## 🎯 Objectives
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Visualize trends and patterns in student performance.
- Identify factors influencing academic success.
- Build a predictive model to estimate student scores.

## 📂 Dataset
- Name: Students Performance in Exams
- Source: Kaggle
- Records: 1,000 students
- Features: Gender, parental education, lunch type, test preparation course, and exam scores.

## 🛠️ Tools and Technologies
- Python
- Google Colab
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- GitHub

## 📊 Project Workflow

## 📈 Key Insights
- Students who completed test preparation scored higher.
- Reading and writing scores show strong positive correlation.
- Parental education influences student performance.
- Socioeconomic factors impact academic outcomes.
